# Introduction

This repository contains various correlations for CPTs.

It is notably used in NGI Digital applications COPIT and SIITE.

# Getting Started

## Development 

### Dependencies

- poetry

### Installing

```bash
poetry install
```

then activate the virtual environment:

```bash
poetry shell
```

### Nixos

If you use nix flakes, you can instead use:

```bash
nix develop .
```

## Run the tests

```bash
pytest-watcher ./
```
